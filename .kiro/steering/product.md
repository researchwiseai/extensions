# Product Overview

Pulse is an AI-powered text analysis platform that provides qualitative research
tools for Microsoft Excel and Google Sheets. The product enables users to
analyze unstructured text data (surveys, feedback, comments) directly within
spreadsheet environments.

## Core Features

- **Sentiment Analysis**: Evaluate emotional tone of text entries
- **Theme Generation**: AI-driven discovery of topics and patterns in text data
- **Theme Allocation**: Assign relevant themes to text entries automatically
- **Theme Management**: Save, reuse, and organize custom theme sets
- **Summarization**: Condense text entries into concise summaries
- **Extract Named Entities**: Designed to find product names within larger text

## Target Users

- Researchers conducting qualitative analysis
- Customer experience teams analyzing feedback
- Market researchers processing survey data
- Academic researchers coding qualitative data

## Integration Points

- Microsoft Excel add-in with Office.js APIs
- Google Sheets add-on using Apps Script
- ResearchWise AI backend for text processing
- OAuth2 authentication for secure account connection

## Key Value Proposition

Eliminates manual coding and subjective bias in qualitative analysis by
providing consistent, AI-driven text classification tools within familiar
spreadsheet environments.
